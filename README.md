# Companion API

## .env file structure


```
# Google credentials
GOOGLE_API_KEY=
GOOGLE_CSE_ID=
GOOGLE_CREDENTIALS_ID=

# LLM path
MODEL_PATH=

# MongoDB credentials, leave out if you want to connect to localhost
BASTION_HOST=
BASTION_USER=
PRIVATE_KEY_PATH=
MONGO_HOST=
LOCAL_MONGO_PORT=
```

## To run server, run:

```
python manage.py runserver
```
